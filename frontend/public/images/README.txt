รูปตัวละคร (Role) เกมหมาป่า
============================

วางไฟล์ .png ในโฟลเดอร์นี้ ตั้งชื่อให้ตรงชื่อ role (ตัวแรกพิมพ์ใหญ่)
ระบบโหลดอัตโนมัติ ใช้เป็นการ์ด role / avatar ที่นั่ง / ไอคอนต่างๆ

ไฟล์ที่ใช้:
  Werewolf.png    หมาป่า
  WolfCub.png     ลูกหมาป่า   ← ใหม่ (ถ้าไม่มี ระบบใช้ Werewolf.png แทนอัตโนมัติ)
  Seer.png        ผู้หยั่งรู้
  Doctor.png      หมอ
  Villager.png    ชาวบ้าน
  Fool.png        คนบ้า
  Hunter.png      นายพราน
  Witch.png       แม่มด
  Cupid.png       คิวปิด

คำแนะนำ:
- สัดส่วนจัตุรัส ~400x400px ขึ้นไป (จะถูก crop แบบ cover) พื้นหลังทึบได้ (การ์ด)
- ไฟล์ .png ชื่อตรงเป๊ะ ตัวแรกพิมพ์ใหญ่: WolfCub.png

** รูปลูกหมาป่า (WolfCub.png) — รายละเอียด **
  • สื่อ "ลูกหมาป่า / หมาป่าตัวเล็ก น่ากลัวแบบน่ารัก" ให้ดูต่างจากหมาป่าโต
  • โทนแนะนำ: ส้ม-แดง (#ff7b4b) เพื่อให้เข้ากับสีบทบาทในเกม
  • ใช้ที่: การ์ดตอนเปิดบทบาท / avatar ที่นั่ง / ไอคอน HUD / หน้าจบเกม /
    ปุ่มเปิดในล็อบบี้ / แกลเลอรีบทบาท — จุดเดียวครบทุกที่
  • ถ้ายังไม่มีไฟล์นี้ เกมเล่นได้ปกติ: ทุกจุดจะ fallback ไปใช้ Werewolf.png
    อัตโนมัติ (ไม่มีรูปแตก) — ใส่เมื่อไรก็เห็นทันที

หมายเหตุ: รูป "ฉาก/พื้นหลังโต๊ะ" ของเกมหมาป่า อยู่ที่โฟลเดอร์ werewolf/
