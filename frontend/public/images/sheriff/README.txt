รูปการ์ด Sheriff of Nottingham
================================

วางไฟล์รูป (.png) ในโฟลเดอร์นี้ โดยตั้งชื่อไฟล์ให้ตรงกับ "ชนิดสินค้า" ด้านล่าง
การ์ดจะโหลดรูปอัตโนมัติ ถ้าไม่มีไฟล์จะแสดง emoji แทน (ไม่ error)

ไฟล์ที่ต้องมี (8 ใบ):

  สินค้าถูกกฎหมาย
    apple.png      🍎 แอปเปิล
    cheese.png     🧀 ชีส
    bread.png      🍞 ขนมปัง
    chicken.png    🐔 ไก่

  ของเถื่อน
    pepper.png     🌶️ พริกไทย
    mead.png       🍺 เหล้าหมัก
    silk.png       🧵 ผ้าไหม
    crossbow.png   🏹 หน้าไม้

  หลังการ์ด (ไพ่คว่ำ) — ใช้กับ กองจั่ว / มือคนอื่น / ถุงพ่อค้าตอนตรวจ
    back.png       🎴 ลายหลังการ์ด (ถ้าไม่มี จะใช้ลายทแยงสีเข้มแทน)

คำแนะนำ (การ์ด):
- การ์ดหน้า (8 ใบ): สัดส่วนแนวตั้ง ~3:4 หรือ 1:1 (crop แบบ object-fit: cover)
  ขนาดแนะนำ ~300x400px ขึ้นไป (คมชัดบนจอ retina)
- หลังการ์ด (back.png): สัดส่วนแนวตั้ง ~3:4 เช่นกัน ลายควรเต็มใบ (จะถูก crop)
- ชื่อไฟล์ตรง "ชนิด" (id) เป๊ะ ตัวพิมพ์เล็กทั้งหมด นามสกุล .png


================================================================
รูปพื้นผิวโต๊ะ (ไม่บังคับ — มีลาย CSS สำรองอยู่แล้ว)
================================================================
ถ้าใส่ไฟล์เหล่านี้ โต๊ะจะสวยขึ้น ถ้าไม่ใส่ ระบบใช้ลายไล่สี CSS แทนอัตโนมัติ
(ไม่มีรูปแตก) — เป็น "ของแถม" ล้วนๆ

  table-wood.png   พื้นผิวไม้ของโต๊ะ (ปูทั้งพื้นหลังโต๊ะ)
                   • แนะนำแบบ tileable/seamless หรือภาพใหญ่ ~1200x800px
                   • โทนน้ำตาลเข้ม (จะถูก blend แบบ overlay กับลายไม้ CSS)

  felt.png         พื้นผิวสักหลาด/ผ้าปูตรงกลางโต๊ะ (โซนเล่น)
                   • โทนเขียวเข้ม ~800x500px (blend แบบ soft-light)

หมายเหตุเทคนิค: 2 ไฟล์นี้ใส่เป็น background-image ซ้อนบนลาย CSS
ถ้าไฟล์ไม่มี เบราว์เซอร์ข้ามเลเยอร์นั้นให้เอง → เห็นลาย CSS ปกติ

(เหรียญทองในเกมวาดด้วย CSS อยู่แล้ว ไม่ต้องมีรูป — แต่ถ้าอยากเปลี่ยนภายหลัง
 บอกได้ จะเพิ่มจุดโหลด coin.png ให้)

  bribe.png      🤝 สินบน

  pass.png       👋 ปุ่มปล่อยผ่าน

หมายเหตุ: ทุกไอคอนวางในกรอบ "เหรียญ" ครีมอัตโนมัติ (พื้นขาวของรูปจะกลืน)
ถ้ารูปพื้นหลังโปร่งใสอยู่แล้ว ก็แสดงสวยบนเหรียญเช่นกัน

คำแนะนำ (ไอคอน):
- สี่เหลี่ยมจัตุรัส 1:1 พื้นหลังโปร่งใส (PNG มี alpha)
- ขนาด ~64x64px หรือ 128x128px (แสดงจริง 14–30px)
- object-fit: contain (ไม่ถูก crop) — วางกลางภาพได้เลย

ชนิด (id) เหล่านี้กำหนดใน backend/games/sheriff/cards.js — ถ้าเพิ่มสินค้าใหม่
ให้ตั้งชื่อไฟล์ตาม id ใหม่นั้น
