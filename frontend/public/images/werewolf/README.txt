รูปประกอบฉากเกมหมาป่า (โต๊ะวงหมู่บ้าน)
========================================

รูปทั้งหมดในโฟลเดอร์นี้ "ไม่บังคับ" — ถ้าไม่ใส่ ระบบใช้ลายไล่สี/แสง CSS แทน
อัตโนมัติ (ไม่มีรูปแตก ไม่ error) ใส่เพื่อให้ฉากสวยขึ้นเป็น "ของแถม" ล้วนๆ

หมายเหตุ: รูป "ตัวละคร" (Werewolf.png, Seer.png, ฯลฯ) อยู่ที่โฟลเดอร์แม่
  /images/  ไม่ใช่ที่นี่ — โฟลเดอร์นี้ไว้สำหรับ "ฉาก/พื้นหลัง" เท่านั้น


────────────────────────────────────────────────────────────────
1) พื้นหลังท้องฟ้า (ปูเต็มโต๊ะ, ระบบ cross-fade กลางวัน↔กลางคืนให้เอง)
────────────────────────────────────────────────────────────────

  night-sky.png   ท้องฟ้ากลางคืน (ใช้ตอน NIGHT / NIGHT_WITCH / HUNTER)
                  • โทนน้ำเงิน-ม่วงเข้ม มีดาว/หมอก/แสงจันทร์
                  • วางทับลายดาว CSS อีกที (ถ้าใส่รูปเองก็เห็นทั้งคู่)

  day-sky.png     ท้องฟ้า/บรรยากาศกลางวัน (ใช้ตอน DAY / VOTING)
                  • โทนอบอุ่น (รุ่งเช้า/สนธยา) น้ำตาลทอง–ฟ้าอ่อน

  คำแนะนำ: แนวนอน ~1200x760px, ขอบมนไม่ต้อง (โต๊ะ crop ให้)

  ** เรื่องตัวหนังสืออ่านไม่ออก (สำคัญ) **
  ตอนนี้แอปใส่ "ชั้นมืดโปร่ง" คลุมรูปพื้นหลังให้อัตโนมัติแล้ว + มีฉากมืด
  หลังข้อความกลาง → ตัวหนังสือสีอ่อนอ่านออกกับรูปเกือบทุกแบบ
  แต่เพื่อผลลัพธ์ดีที่สุด:
    • ใช้รูปโทน "เข้ม/มืด" (กลางคืน = น้ำเงินเข้ม, กลางวัน = สนธยาอุ่นเข้ม)
    • เลี่ยงรูปสว่างจ้า/คอนทราสต์สูงตรงกลางภาพ (ตรงที่ข้อความอยู่)
    • ถ้ายังสว่างไป ให้หรี่ความสว่างรูปลง ~30-40% ก่อนบันทึก


────────────────────────────────────────────────────────────────
2) ไฟกลางวง (hearth) — จุดศูนย์กลางของหมู่บ้าน
────────────────────────────────────────────────────────────────
วางไว้กลางวง หลังข้อความสถานะ ถ้าไม่ใส่จะเป็นแสงเรืองกลมๆ แทน

  campfire.png    กองไฟ (โชว์ตอนกลางคืน) — มี animation กระพริบไฟให้
                  • พื้นหลังโปร่ง (transparent PNG) สำคัญมาก
                  • สัดส่วนจัตุรัส ~400x400px วัตถุอยู่กลางภาพ

  well.png        บ่อน้ำ/ป้ายหมู่บ้าน (โชว์ตอนกลางวัน)
                  • พื้นหลังโปร่ง จัตุรัส ~400x400px

  ระบบวางในกรอบ ~210x210px แบบ contain (ไม่ครอป) จัดกลางอัตโนมัติ

เคล็ดลับ
────────────────────────────────────────────────────────────────
- ชื่อไฟล์ต้องตรงเป๊ะ ตัวพิมพ์เล็ก .png (night-sky.png, day-sky.png,
  campfire.png, well.png)
- PNG โปร่งพื้นหลังสำหรับ campfire/well เท่านั้น; ท้องฟ้าเป็นรูปทึบได้
- ใส่ทีละไฟล์ได้ ไม่ต้องครบ — อันไหนไม่มีก็ใช้ CSS สำรอง
- หลังใส่รูป: refresh หน้าเว็บก็เห็นเลย (ไม่ต้อง build ใหม่ ถ้ารันผ่าน dev/preview)
